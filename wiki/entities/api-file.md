# API File

> **Categoria:** Marketplace de APIs | **Aparece em:** 1 fonte

## O que é

Segundo [[lucas-garcia-pit]], API File é descrita como "a maior loja de API da história". Funciona como marketplace de APIs que inclui scraping do Google Maps, integrável ao Claude Code via chave de API.

> ⚠️ **Nota de verificação**: A descrição "maior loja de API da história" é afirmação do criador, não verificada independentemente. Pode ser exagero de marketing.

## Configuração com Claude Code

1. Acessar API File → Settings → API e integrações
2. Copiar a chave de API
3. No Claude Code: colar a chave e solicitar uso da API

## Capacidade documentada

- Scraping do Google Maps
- Enriquecimento com busca adicional na internet sobre cada lead
- Geração de links de WhatsApp diretos na tabela de leads

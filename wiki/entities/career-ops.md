# Career Ops

> **Categoria:** Ferramenta open-source | **Licença:** MIT | **Aparece em:** 1 fonte

## O que é

Sistema de busca de emprego construído com Claude Code por desenvolvedor independente. Transforma a busca de emprego de esforço individual para pipeline automatizado e estruturado.

## Funcionalidades

- Escaneia páginas de carreira de empresas automaticamente
- Avalia vagas em **10 dimensões** de compatibilidade
- Reescreve currículo otimizado para **ATS** por vaga específica
- Dashboard terminal para rastreamento de candidaturas
- Prepara conteúdo de entrevista
- Mantém camada de decisão humana ("aplica só onde há match real")
- Integra com múltiplas empresas
- Gera PDFs otimizados

## Resultado documentado

700+ vagas avaliadas → oferta de emprego real obtida.

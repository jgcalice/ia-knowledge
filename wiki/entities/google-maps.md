# Google Maps

> **Categoria:** Plataforma | **Fabricante:** Google | **Aparece em:** 3 fontes

## Papel no wiki

Google Maps aparece em **dois papéis distintos e complementares**:

### Papel 1: Fonte de dados para geração de leads B2B (scraping)

Dois criadores independentes usam o Maps como origem de dados para prospecção, com ferramentas diferentes:

- [[lucas-garcia-pit]] usa [[api-file]] para scraping do Maps
- [[hudson-brendon]] usa [[apify]] para scraping do Maps

### Papel 2: Mercado de oportunidades para agência de SEO local (prompts)

[[derek-gray]] usa o Maps de forma ortogonal: em vez de coletar dados de negócios para vender algo a eles, ele **identifica negócios rankeados #6–20** (invisíveis para ~70% dos clientes) e os ajuda a subir para o top 3 cobrando $500–$1K/mês de retainer. Aqui o Maps é simultaneamente o **canal de prospecção** e o **canal de entrega do serviço**.

## Dados disponíveis via scraping

- Nome do estabelecimento
- Telefone
- Endereço e cidade
- Site
- Redes sociais
- Avaliação do Google
- Categoria de negócio
- **E-mail: raramente disponível** (limitação conhecida, mencionada por Hudson Brendon)

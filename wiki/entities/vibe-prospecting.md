# Vibe Prospecting

> **Categoria:** Ferramenta — Conector para Claude.ai | **Aparece em:** 1 fonte

## O que é

Vibe Prospecting é uma ferramenta de geração de leads disponível como **Conector nativo no Claude.ai**. Diferente de ferramentas como [[apify]] (que usam MCP/CLI) ou [[api-file]] (marketplace de APIs), o Vibe Prospecting se integra diretamente pela interface web do Claude, sem necessidade de instalar Claude Code ou configurar ambiente técnico.

## Como funciona

1. Usuário acessa **Personalizar → Conectores** no Claude.ai
2. Instala os dois conectores do Vibe Prospecting via marketplace
3. Faz login nos dois conectores
4. Usa um prompt estruturado para especificar nicho, cargos e cidade
5. Claude retorna uma lista de leads em formato de planilha

## Diferenciais em relação a outras ferramentas de leads

| Critério | Vibe Prospecting | [[apify]] | [[api-file]] |
|----------|-----------------|-----------|--------------|
| Interface | Web (Claude.ai) | CLI/MCP | CLI/MCP |
| Custo de acesso | Gratuito | Varia | Varia |
| Barreira técnica | Baixa | Média | Média |
| Método de integração | Connector nativo | MCP/conector | API key |

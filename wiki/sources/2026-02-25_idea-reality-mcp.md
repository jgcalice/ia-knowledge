# Ferramenta GitHub Avalia Originalidade de Ideias de IA

> **Fonte:** [[2026-02-25_michael_kocher_ai_automation_DVMzau6kW_k]] | **Autor:** @Michael Kocher | AI Automation | **Data:** 2026-02-25 | **Formato:** Reel (66s) | **[↗ Ver post](https://www.instagram.com/reel/DVMzau6kW_k/?igsh=NjNiZXRjMWZicWow)**

## TL;DR

O GitHub lançou o Idea Reality MCP, ferramenta que pontua ideias de IA de 0 a 100 com base em projetos existentes no GitHub, Hacker News, NPM e outras plataformas.

## Contexto

Michael Kocher apresenta uma ferramenta nova do GitHub que resolve um problema clássico de founders: descobrir se a ideia já existe antes de construir. O problema relatado é founders gastando 6 meses em projetos com 12 alternativas open-source já existentes.

## O que foi ensinado

- **Idea Reality MCP**: descreva o produto, ele busca em GitHub, Hacker News, NPM e retorna um score de 0-100
- Score baixo (ex: 14) = ideia saturada, muitas implementações existentes
- Score alto (ex: 73) = nicho específico com menos concorrência
- Exemplo testado: "AI agent para resumir Slack" → score 14 (saturado)
- Exemplo testado: "gerar SQL de linguagem natural para gestão de estoque" → score 73 (nicho viável)
- A ferramenta automatiza a pesquisa que fundadores deveriam fazer antes de codificar

## Insights para o wiki

- Validação de ideias com IA antes do desenvolvimento é um padrão emergente
- Foco em nichos verticais específicos é mais promissor do que soluções genéricas
- O conceito de "reality check antes de escrever código" tem valor de negócio claro

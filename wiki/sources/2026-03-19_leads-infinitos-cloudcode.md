# Leads Infinitos com CloudCode

> **Fonte:** [[2026-03-19_lucas_garcia_pit_ia_negocios_DWFNti0D-3u]] | **Autor:** @Lucas Garcia Pit | IA & Negócios | **Data:** 2026-03-19 | **Formato:** Reel (134s) | **[↗ Ver post](https://www.instagram.com/reel/DWFNti0D-3u/?igsh=ZzVsaGY2eXRraTdu)**

## TL;DR

Use o Claude Code com a [[api-file]] para fazer scraping do Google Maps e gerar listas de leads enriquecidos — com links de WhatsApp prontos — de forma ilimitada e automatizada.

## Contexto

Lucas Garcia Pit ensina em 134 segundos como usar o Claude Code integrado à API File (descrita como "a maior loja de API da história") para captar leads estratégicos do Google Maps. O diferencial do método é o enriquecimento dos dados: além de coletar o lead, o Claude busca contexto adicional na internet para personalizar abordagens.

## O que foi ensinado

- **Configuração**: Acessar API File → Settings → API e integrações → copiar chave → colar no Claude Code
- **Estratégia antes do scraping**: Dar contexto completo ao Claude sobre o projeto, o ICP (Ideal Customer Profile) e quem é o lead ideal — quanto mais contexto, leads mais qualificados
- **Enriquecimento**: Pedir ao Claude para buscar na internet dados adicionais sobre cada estabelecimento encontrado no Google Maps
- **Praticidade**: Solicitar que a tabela gerada já inclua links diretos para WhatsApp de cada lead
- **Escalabilidade**: O processo pode ser repetido indefinidamente; usar o mesmo chat evita repetição de leads já capturados

## Insights para o wiki

- Dois criadores (Lucas e [[hudson-brendon]]) ensinam métodos de geração de leads do Google Maps com Claude — mas com ferramentas diferentes ([[api-file]] vs [[apify]]). Abordagens convergentes.
- O ponto chave é: **contexto sobre ICP determina qualidade dos leads**, não apenas o volume
- Links de WhatsApp pré-gerados na tabela = redução de fricção no processo de contato

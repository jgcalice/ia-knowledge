# Certificação Gratuita Claude Certified Architect

> **Fonte:** [[2026-03-26_bashiri_become_an_ai_engineer_DWW9dI_J3Gb]] | **Autor:** @Bashiri | Become an Ai Engineer | **Data:** 2026-03-26 | **Formato:** Reel (44s) | **[↗ Ver post](https://www.instagram.com/reel/DWW9dI_J3Gb/?igsh=MXU1MDJxOWJpMXZtZw==)**

## TL;DR

A Anthropic oferece gratuitamente a certificação "Claude Certified Architect" — 2 horas, 60 questões, proctored com webcam — especialmente valiosa para engenheiros de software migrando para IA.

## Contexto

Bashiri cria conteúdo sobre transição de carreira para AI Engineering. A certificação é descrita como equivalente a uma AWS Certification, mas para IA, e mencionada como algo que a Deloitte paga para funcionários obterem.

## O que foi ensinado

- **Nome da certificação:** Claude Certified Architect
- **Formato:** 2 horas, 60 questões, proctored (webcam obrigatória)
- **Conteúdo:** construção de agentes e configuração em pipeline de deployment
- **Custo:** gratuito
- **4 passos para obter:**
  1. Entrar no Claude Partner Network (gratuito)
  2. Fazer os prep courses
  3. Fazer o practice exam
  4. Agendar o exame real
- **Público-alvo:** especialmente engenheiros de software transitando para AI Engineering

## Insights para o wiki

- Certificação gratuita com peso corporativo (Deloitte) é diferencial de carreira concreto
- AI Engineering como trilha de carreira está se consolidando com certificações formais

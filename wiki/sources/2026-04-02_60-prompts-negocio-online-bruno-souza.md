# 60 Prompts para Construir Negócio Online de 6 Dígitos com Claude

> **Fonte:** [[2026-04-02_bruno_souza_DWowtrKE7r9]] | **Autor:** @Bruno Souza | **Data:** 2026-04-02 | **Formato:** Carousel (10/13 slides processados) | **[↗ Ver post](https://www.instagram.com/p/DWowtrKE7r9/?img_index=12&igsh=bzN1Yjh0ZWYzMm82)**

## TL;DR

60 prompts organizados em 10 categorias para usar Claude na construção de negócio online — escrita humanizada, estratégia de conteúdo, SEO, criação de SaaS, marketing/vendas, Claude Code, produtividade, debugging, landing pages e copy.

## Contexto

Segunda ingestão de Bruno Souza (@real_bruno_souza). Aqui ele muda o foco do LinkedIn para a construção de negócio online em si — uma coleção extensa de prompts por categoria. Título original: "R.I.P. ChatGPT — 60 Claude Prompts to Build a 6-Figure Online Business".

## O que foi ensinado (por categoria)

### 1. Writing Like a Human (6 prompts)
- Analisar 5 peças de escrita e extrair a voz do autor
- Reescrever texto gerado por IA para soar natural
- Escrever thread no Twitter com regras específicas
- Transformar características de produto em copy focado em benefícios
- Escrever história pessoal para LinkedIn com estrutura definida
- Criar guia de voz para uma marca

### 2. Content Strategy (6 prompts)
- Calendário de conteúdo de 30 dias para nicho/audiência específicos
- Transformar blog em threads (Twitter), posts (LinkedIn) e captions (Instagram)
- 20 ganchos para tópico com técnicas de curiosidade e estatísticas
- Análise dos 10 melhores posts para extrair padrões
- Roteiro para vídeo do YouTube
- 10 ideias de desmistificação para nicho

### 3. SEO & Growth (6 prompts)
- Resumo de conteúdo completo para palavra-chave (TAM, intenção, estrutura)
- Análise de 5 páginas concorrentes para gap de conteúdo
- Estratégia de SEO programático para 100+ páginas
- Landing page que rankeia para palavra-chave específica
- Auditoria de estrutura de site para problemas de SEO
- Mapa de autoridade topical para nicho

### 4. SaaS Building (4 prompts)
- Validação de ideia de SaaS com análise de mercado, competição e distribuição
- Página de preços com 3 tiers
- E-mails de integração (onboarding) para produto
- Plano de integração Stripe

### 5. Marketing & Sales (6 prompts)
- Iscas digitais para nicho/produto
- Sequência de emails frios com estrutura PAS
- Objeções ao produto com contra-argumentos
- Loop viral para recrutamento de usuários via referência
- Títulos de landing page com múltiplas estruturas de copywriting
- Programa de referência com incentivos e rastreamento

### 6. Claude Code & CLI (6 prompts)
- Revisão de código + arquivo CLAUDE.md com padrões
- Simulação de code review criticando um git diff
- Análise de stack trace + sugestão de correção
- Refatoração para legibilidade e testabilidade
- Geração de testes completos para uma função
- Revisão de rota de API para vulnerabilidades

### 7. Produtividade & Sistemas (6 prompts)
- Workflow semanal para 2 funções
- Lista de tarefas → plano de execução de 7 dias com delegação
- 10 automações de IA para economizar 5h/semana
- Matriz de decisão com avaliação de risco
- Sistema de batching: 30 dias de conteúdo em 1 sessão de 8h
- Sistema pessoal com rotina matinal e estrutura de metas

### 8. Debugging & Refactoring (6 prompts)
- Refatorar código funcional mas impossível de manter
- 10 passos diagnósticos para erro em produção
- Comparar duas abordagens de implementação
- Plano de migração tecnológica sem downtime
- Revisão de schema de banco (índices faltantes)
- Estratégia de error handling abrangente

### 9. Landing Pages & Copy (6 prompts)
- Cópia above-the-fold com título, subtítulo e CTA
- E-mails para solicitar depoimentos de clientes
- Página de preços com FAQ
- Template de estudo de caso
- Wireframe de homepage em texto
- Microcópia (estados vazios, confirmações de sucesso)

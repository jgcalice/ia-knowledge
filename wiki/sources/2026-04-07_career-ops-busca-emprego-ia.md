# Career Ops: Sistema de Busca de Emprego com IA

> **Fonte:** [[2026-04-07_technology_DW1R1LOjaNS]] | **Autor:** @Technology | **Data:** 2026-04-07 | **Formato:** Carousel (2 slides) | **[↗ Ver post](https://www.instagram.com/p/DW1R1LOjaNS/?img_index=3&igsh=a2o4a3BjaTZnN2Np)**

## TL;DR

Um desenvolvedor construiu o Career Ops com Claude Code — sistema que avaliou 700+ vagas, automatizou candidaturas com currículos otimizados para ATS e resultou em oferta de emprego real.

## Contexto

Post compartilha um projeto do Reddit (r/ClaudeAI) onde um desenvolvedor criou uma ferramenta open-source (licença MIT, GitHub: santifer/career-ops) que transforma a busca de emprego em pipeline estruturado.

## O que foi ensinado

- **Career Ops** converte busca de emprego de esforço individual para sistema automatizado
- Funcionalidades:
  - Escaneia páginas de carreira de empresas
  - Avalia vagas em **10 dimensões** de compatibilidade real
  - Reescreve currículo otimizado para ATS por vaga
  - Rastreia candidaturas (dashboard terminal)
  - Prepara conteúdo de entrevista
  - Mantém camada de decisão humana ("aplica só onde há match real")
  - Integra com múltiplas empresas
  - Gera PDFs otimizados
- **700+ vagas avaliadas**, resultando em oferta concreta
- Ferramenta gratuita e open-source (MIT)

## Insights para o wiki

- Busca de emprego como "pipeline de dados" é mudança de paradigma — qualidade sobre quantidade
- ATS (Applicant Tracking System) é barreira crítica: currículo genérico é filtrado antes de humano ver
- Integra com [[carreira-com-ia]] e [[2026-04-11_transformacao-linkedin-ia]] — otimizar LinkedIn + automatizar busca = estratégia completa
- Claude Code usado para construir ferramenta de carreira, não apenas gerar conteúdo

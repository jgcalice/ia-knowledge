# SEO na Primeira Página do Google com 3 Arquivos de Texto

> **Fonte:** [[2026-04-11_brycen_wood_DW-ZK8bDGNf]] | **Autor:** @Brycen Wood | **Data:** 2026-04-11 | **Formato:** Reel (51s) | **[↗ Ver post](https://www.instagram.com/reel/DW-ZK8bDGNf/?igsh=ajd2eHJ5ZDJ1Ymhw)**

## TL;DR

Três arquivos de texto adicionados ao site em 10 minutos colocaram o negócio na primeira página do Google sem anúncios — usando llms.txt, Markdown mirrors e sitemap.

## Contexto

Brycen Wood, dono de pequeno negócio (Sum­mit Wraps and Graphics), compartilha resultado prático de estratégia de SEO para IA usando Claude Code para criar os arquivos.

## O que foi ensinado

- **Arquivo 1 — llms.txt**: descreve exatamente o que o negócio faz para os modelos de IA entenderem
- **Arquivo 2 — Markdown mirrors**: versão limpa das páginas do site que a IA consegue ler sem ruído de código/CSS
- **Arquivo 3 — Sitemap limpo**: informa ao Google quais páginas são importantes
- Resultado: primeira página do Google em 1 mês, sem pagar por anúncios
- Prompts do Claude Code usados disponíveis no site do autor

## Insights para o wiki

- llms.txt está emergindo como padrão de SEO para IA — análogo ao robots.txt
- Pequenos negócios físicos podem competir com grandes players usando IA no SEO
- Conecta-se ao [[markitdown]] — Markdown como formato de legibilidade para IA é tema recorrente

# Criando Listas de Prospecção com Claude e Vibe Prospecting

> **Fonte:** [[2026-04-14_eduardo_santos_ia_nas_empresas_DXIO1mpCbzU]] | **Autor:** @Eduardo Santos | IA nas empresas | **Data:** 2026-04-14 | **Formato:** Reel (35s) | **[↗ Ver post](https://www.instagram.com/reel/DXIO1mpCbzU/?igsh=NTA3NHB1YjZwcjll)**

## TL;DR

Usando os Conectores nativos do Claude.ai com o [[vibe-prospecting]], é possível gerar listas de leads por nicho, cargo e cidade de forma gratuita e diretamente no navegador — sem Claude Code, sem APIs pagas.

## Contexto

Eduardo Santos (@Eduardo Santos | IA nas empresas) ensina como usar a interface web do Claude.ai (não o Claude Code/CLI) para montar listas de prospecção. O diferencial é que o processo usa a funcionalidade de **Conectores** do Claude — uma feature nativa do produto que permite instalar integrações de terceiros diretamente no chat.

## O que foi ensinado

- Acessar o Claude.ai e clicar em **"Personalizar"** → **"Conectores"**
- Clicar no **"+"** para navegar no marketplace de conectores
- Buscar e instalar os dois conectores do **Vibe Prospecting**
- Fazer login nos dois conectores
- Colar o prompt compartilhado pelo autor (disponível via DM ao comentar "LEADS" e seguir o perfil)
- Preencher no prompt: **nicho**, **cargos preferidos** e **cidade**
- Claude gera a lista de leads
- Baixar a planilha gerada e prospectar

## Insights para o wiki

- **Método 4 de geração de leads**: diferente dos métodos anteriores (Apify, API File, Skill), este usa os Conectores nativos do Claude.ai — interface web, gratuita, sem necessidade de instalar Claude Code
- **Conectores como canal novo**: a funcionalidade de Conectores do Claude.ai ainda não havia sido documentada neste wiki — é uma camada diferente de integração comparada ao MCP/CLI
- **Barreira de entrada menor**: o processo é 100% pelo navegador, eliminando configurações técnicas de ambiente
- **Acesso ao prompt via engajamento social**: a técnica de exigir comentário + seguidor para receber o prompt é um padrão de crescimento de canal (já observado em outras fontes)

# Criando uma Marca de R$20.000 em 2 Horas com Claude Code

> **Fonte:** [[2026-04-14_rafael_brandao_DXHFS8uk-su]] | **Autor:** @Rafael Brandão | **Data:** 2026-04-14 | **Formato:** Reel (70s) | **[↗ Ver post](https://www.instagram.com/reel/DXHFS8uk-su/?igsh=amRlZ3R4enUxcjMy)**

## TL;DR

Rafael Brandão demonstra a criação de uma marca completa (logo, paleta, tipografia, landing page, UI kit, social media kits, apresentação comercial, papel timbrado, assinatura Gmail) em menos de 2 horas usando Claude Code com 120+ agentes.

## Contexto

Segunda ingestão de Rafael Brandão. Foco em demonstrar capacidade produtiva do Claude Code para serviços de agência criativa, com valor de mercado estimado em R$20.000.

## O que foi ensinado

- Entregáveis gerados em menos de 2 horas:
  - Logo, paleta de cores, sistema de tipografia
  - Tom de voz, comunicação, estilo visual e fotografia
  - Landing page personalizada
  - UI kit completo
  - Social media kits (Instagram, LinkedIn, YouTube)
  - Apresentação comercial personalizada
  - 2 opções de papel timbrado
  - Assinatura digital para Gmail
- **Arquitetura**: 120+ agentes organizados em times (branding, copy, design)
  - Time de Brand → essência da marca
  - Time de Copy → todos os textos
  - Time de Design → brandbook completo

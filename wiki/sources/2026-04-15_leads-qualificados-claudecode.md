# Leads Qualificados com Claude Code e API

> **Fonte:** [[2026-04-15_flavio_rafael_claude_code_DWnAW7aEVLp]] | **Autor:** @Flávio Rafael | Claude Code | **Data:** 2026-04-15 | **Formato:** Reel (64s) | **[↗ Ver post](https://www.instagram.com/reel/DWnAW7aEVLp/?igsh=MTVwdWd0bGd2cGdudQ==)**

## TL;DR

Skill "lista de alto valor" no Claude Code gera centenas de leads qualificados (nome, email, telefone, WhatsApp, Instagram, LinkedIn) via API, exportando para planilha ou CRM.

## Contexto

Flávio Rafael demonstra workflow de geração de leads qualificados para uso próprio e para clientes, integrando Claude Code com plataforma de API de dados.

## O que foi ensinado

- Skill criada: "lista de alto valor" no Claude Code
- Fluxo:
  1. Entrar na plataforma de API → copiar chave API
  2. Conectar chave no Claude Code
  3. Descrever o tipo de lead desejado
  4. Claude faz perguntas: tipo de empresa, objetivos, regiões, enriquecimento, destino de exportação
  5. Gera fluxograma do processo
  6. Entrega planilha ou integra com CRM
- Dados entregues por lead: nome, email, telefone, WhatsApp, Instagram, LinkedIn
- 3 usos possíveis: campanha Meta Ads (público personalizado/lookalike), prospecção ativa (WhatsApp/ligação), enriquecimento de CRM

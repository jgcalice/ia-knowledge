# 5 Prompts para Transformar seu LinkedIn com IA

> **Fonte:** [[2026-04-16_sanskaar_singh_r_ai_wizard_DXLkANKE24F]] | **Autor:** @Sanskaar Singh R • AI Wizard | **Data:** 2026-04-16 | **Formato:** carousel | **[↗ Ver post](https://www.instagram.com/p/DXLkANKE24F/?img_index=6&igsh=MXV1ODFzeTUzYjIzbQ==)**

## TL;DR

5 prompts para o Claude AI otimizar um perfil de LinkedIn em perspectiva de recrutador, gerando visibilidade real com vagas em menos de uma semana.

## Contexto

Sanskaar Singh Rajput compartilhou um carousel mostrando como deu seu perfil do LinkedIn para o Claude AI e, em uma semana, recrutadores começaram a notá-lo. A diferença central em relação a outros tutoriais é o ângulo do **recrutador como perspectiva do LLM**: o autor não pede ao Claude para "melhorar" o perfil — pede que Claude o avalie como um recrutador senior real faria.

## O que foi ensinado

### Prompt 1: The First Impression Test
```
Here is my LinkedIn profile [Link]. You are a senior recruiter hiring for [your target role]. 
Give me your honest first impression. What makes you want to reach out? 
What makes you click away? Don't sugarcoat it.
```
Objetivo: obter um diagnóstico honesto antes de qualquer reescrita.

### Prompt 2: The Headline Rewrite
```
Rewrite my LinkedIn headline to be keyword-rich, specific, and searchable by recruiters 
hiring for [target role]. Give me 5 versions - from safe and professional to bold 
and attention-grabbing.
```
Entrega: 5 versões do headline em espectro de ousadia (conservador → ousado).

### Prompt 3: The About Section Rewrite
Estrutura: gancho que prende atenção → o que você faz → quem você ajuda → resultados que entrega → CTA simples. Tom humano, não corporativo.

### Prompt 4: The Keywords & Search Prompt
```
Here are 3 job descriptions for roles I want to be hired for. Compare them with my LinkedIn 
profile. Tell me which keywords are missing, which skills I should add, and where exactly 
I should place them so recruiters can find me in searches.
```
Técnica: comparação de JDs com o perfil para identificar lacunas de SEO.

### Prompt 5: The Experience Section Polish
```
Rewrite my experience bullet points using this format: Action verb + what I did + 
measurable result. If I don't have numbers, ask me questions to help me find them. 
Every bullet should sound like an achievement, not a task.
```
Diferencial: se o usuário não tiver métricas, o Claude faz perguntas para ajudá-lo a encontrá-las.

## Insights para o wiki

- **Perspectiva invertida**: enquanto a maioria dos tutoriais pede ao Claude para "melhorar" o texto, Sanskaar pede ao Claude para **simular o julgamento do recrutador** antes de qualquer escrita. Esse diagnóstico primeiro é uma técnica não documentada até aqui.
- **5 versões com espectro de ousadia**: o Prompt 2 entrega um menu de escolha (conservador → ousado), não uma resposta única. Isso devolve a decisão criativa ao usuário.
- **Claude como consultor de métricas** (Prompt 5): se o usuário não tem números, o Claude conduz uma entrevista para descobri-los. Isso transforma o LLM de editor em co-descobridor de conquistas.
- Confirma padrão já documentado em [[2026-04-11_transformacao-linkedin-ia]] ([[bruno-souza]]), com complementos relevantes: abordagem recruiter-first e entrega de múltiplas versões.

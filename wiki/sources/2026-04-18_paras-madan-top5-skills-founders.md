# Top 5 Claude Skills para Fundadores

> **Fonte:** [[2026-04-18_paras_madan_startups_tech_DXQ1hB_iK-s]] | **Autor:** @Paras Madan | **Data:** 2026-04-18 | **Formato:** carousel | **[↗ Ver post](https://www.instagram.com/p/DXQ1hB_iK-s/)**

## TL;DR

Cinco Claude Skills especializadas substituem a contratação de três colaboradores — cada skill cobre um canal de marketing/crescimento diferente para fundadores.

## Contexto

[[paras-madan]] é criador de conteúdo sobre startups e tech. O carousel lista 5 skills prontas distribuídas via `varnam.tech/opendirectory` (diretório aberto de Claude Skills). O argumento central: "drop into Claude Code, save on salaries" — implantar essas skills equivale a ter especialistas dedicados por fração do custo.

## O que foi ensinado

### Skill 01 — Meta Ads
- O agente assume o papel de media buyer especialista
- Navega estruturas de contas de anúncios, identifica picos de CPA, descobre novos públicos
- Transforma respostas brutas da Meta API em relatórios legíveis
- Métricas: ∞ contas de anúncios, 0 logins em dashboard, 1 conversa
- **Por que importa:** fundadores gastam horas em painéis; esta skill transforma tudo em conversa

### Skill 02 — Position Me
- Aponta para um site, faz crawl de páginas e screenshots, audita o posicionamento
- Analisa friction visual + carga cognitiva via modelo LIFT
- Reescreve copy com framework PAS
- Checa GEO (llms.txt, sitemap, JSON-LD schema)
- **Por que importa:** "roda a auditoria que um consultor de $500/hora rodaria — em minutos"

### Skill 03 — LinkedIn Post Generator
- Aceita qualquer input: blog post, press release, uma frase sobre o que foi lançado
- Escolhe automaticamente hook, arco narrativo e formato correto
- Formatos suportados: Founder/Ship (história pessoal), Insight (lição de blog), Product Launch (anúncio de feature)

### Skill 04 — Reddit ICP Monitor
- Monitora subreddits em busca de pessoas descrevendo problemas que a ferramenta resolve
- Pontua cada post de 1 a 5 (relevância ICP); ≥4 dispara elaboração de resposta útil
- Regra de ouro: nunca mencionar o produto a menos que o post solicite recomendações de ferramentas explicitamente
- **Por que importa:** encontra o comprador antes de ele te encontrar

### Skill 05 — Google Trends SEO
- Puxa dados ao vivo da Trends API, gerencia rate limits, cacheia resultados
- Camada 1 (Trends API) + Camada 2 (SEO Workflow): identifica breakout keywords antes do pico
- Constrói outlines de blog SEO-otimizados ao redor das keywords emergentes
- **Por que importa:** "escrever conteúdo sem dados de tendência é cheirar o vento"

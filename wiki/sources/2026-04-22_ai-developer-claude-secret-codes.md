# 13 Pseudo-Comandos Secretos do Claude — Ativadores de Modo com Prefixo /

> **Fonte:** [[2026-04-22_ai_developer_code_ai_js_DXcR5N0iTGy]] | **Autor:** @AI Developer | Code + AI | JS | **Data:** 2026-04-22 | **Formato:** imagem única | **[↗ Ver post](https://www.instagram.com/p/DXcR5N0iTGy/)**

## TL;DR

13 pseudo-comandos com prefixo "/" que funcionam como ativadores semânticos informais no Claude, comunicando o modo de resposta desejado antes do prompt.

## Contexto

Post de imagem única atribuído originalmente a @ai_with_twinkle e republicado por @AI Developer | Code + AI | JS. O conteúdo apresenta uma lista de "secret codes" para Claude, formatados com barra (/) como se fossem comandos nativos. Na prática, são ativadores semânticos com notação de slash — não comandos reais do Claude Code CLI.

## O que foi ensinado

Os 13 pseudo-comandos documentados:

| Comando | Efeito declarado |
|---------|-----------------|
| `/godmode` | Modo agressivo e poderoso — maximiza impacto |
| `/devil` | Apresenta a visão oposta à do usuário ("steelman" da oposição) |
| `/10x` | Reescreve qualquer texto 10x mais conciso e afiado |
| `/pitch` | Gera pitch de 30 segundos para investidor ou cliente |
| `/ghost` | Resposta human-like, menos robótica |
| `/compare` | Análise lado-a-lado de duas ou mais opções |
| `/scout` | Identifica riscos e pontos cegos no plano |
| `/artifacts` | Constrói apps ao vivo no chat (via Claude Artifacts) |
| `/ooda` | Aplica o loop OODA para resolver problemas complexos de código |
| `/critique` | Encontra falhas e melhora o que foi apresentado |
| `/explainlikeim5` | Explicação ultra-simples (ELI5) |
| `/brief` | Resposta mais curta possível, sem rodeios |
| `/teacher` | Modo mentor — debate e orientação |

## Insights para o wiki

- Cria uma nova subcategoria de ativadores semânticos: a notação com "/" como interface de seleção de modo, distinta das palavras-gatilho sem prefixo ([[castilhoia]]) e dos comandos reais do Claude Code ([[sal-shirgaleev]])
- Vários comandos sobrepõem ativadores já documentados: `/devil` ≈ Devil's Advocate, `/scout` ≈ Red Team, `/explainlikeim5` = ELI5 ([[adriano-couto]])
- `/artifacts` pode referenciar o recurso real de Artifacts do Claude.ai; `/ooda` é uma referência ao OODA loop (Observe → Orient → Decide → Act) — o único com fundamentação metodológica explícita
- O formato de lista mnemonicável facilita adoção sem necessidade de entender o mecanismo subjacente

> ⚠️ Esses comandos são **informais** — não são comandos nativos do Claude Code CLI. Funcionam como ativadores semânticos com prefixo "/" mas dependem de o modelo interpretar a intenção do slash. Contrastar com comandos reais: [[2026-04-22_sal-shirgaleev-5-comandos-claude]]
